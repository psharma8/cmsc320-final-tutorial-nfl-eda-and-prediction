This is a final tutorial project for Data Science class at UMD.

Topics covered in this tutorial :
1) Data Wrangling
2) EDA
3) linear and logistic regression

The data used for this project are taken from https://github.com/ryurko/nflscrapR-data

files in regular_season contain nfl play by play detail of each game for regular season from 
1st week of 2009 to 11th week of 2018.
